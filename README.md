    Crud
</h1>

### Descrição
Um crud feito em Node.js e com front em React.js


### Executando
1. Clone o projeto com o Git.

```bash
$ git clone https://github.com/felipesgt/crud-carros.git
```

2. Executando o Crud:
### Server

```bash

# Entre na pasta server
$cd server
#Depois instale as dependencia com yarn
$yarn
#Rode o servidor
$yarn start
```

### Web

```bash
# Entre na pasta
$ cd server
# Start the client
$ yarn web start
```
